# PhotoGalleryService

クラウド対応画像ストアアプリ：PhotoGalleryを公開サービスとするためのソースをコミットするためのリポジトリです。
Spring の数あるサービスのうち、
・SpringBoot
・SpringSecurity
・SpringSession
・SpringDataJDBC
・SpringDataJPA
などを軽微に使用しています。
